# this-person-does-not-exist

这是一个基于深度学习技术的项目，生成虚拟人物的照片，这些照片看似真实但实际上是不存在的人物。代码实现了生成对抗网络（GAN）的模型，能够创建逼真的人脸图像。

## 使用方法
```
python3 main.py
```

## 数据参考（直接下载）
 - 10万张脸（ https://github.com/ozgrozer/100k-faces ）
